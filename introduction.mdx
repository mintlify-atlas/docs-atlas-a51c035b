## Why Trieve?

Trieve combines the best search technologies into a single, easy-to-use API:

## Key Features

<AccordionGroup>
  <Accordion title="Advanced Search Capabilities" icon="search">
    - **Hybrid Search**: Combine semantic and full-text search for best results
    - **Cross-Encoder Re-ranking**: Use BAAI/bge-reranker-large for optimal relevance
    - **Typo Tolerance**: Neural sparse-vector search handles misspellings
    - **Filtering**: Date-range, substring, tag, numeric, and custom filters
    - **Recency Biasing**: Prevent stale results by boosting recent content
  </Accordion>

  <Accordion title="Recommendations & RAG" icon="robot">
    - **Similarity Recommendations**: Find related content based on user behavior
    - **Managed RAG**: Topic-based memory management for conversational AI
    - **Custom Context RAG**: Select your own context for generation
    - **Any LLM**: Use OpenRouter to access any large language model
  </Accordion>

  <Accordion title="Content Organization" icon="folder">
    - **Grouping**: Organize chunks by file to prevent duplicate top-level results
    - **Metadata**: Rich metadata support for filtering and organization
    - **Tracking IDs**: External system integration with custom identifiers
    - **Bulk Operations**: Upload up to 120 chunks at once
  </Accordion>

  <Accordion title="Deployment Options" icon="server">
    - **Cloud Hosted**: Start with 1,000 free chunks on our managed platform
    - **Self-Hosted**: Deploy in your VPC or on-premise
    - **Docker Compose**: Quick local development setup
    - **Kubernetes**: Production-ready helm charts for AWS, GCP, and more
  </Accordion>
</AccordionGroup>
